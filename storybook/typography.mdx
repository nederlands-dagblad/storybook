## Font weights

Our typography system uses various font weights to create visual hierarchy and emphasis:

- **Light (300)**: Used for body text and less emphasized content
- **Regular (400)**: Standard weight for most text
- **Semibold (600)**: Used for subheadings and slightly emphasized text
- **Bold (700)**: Used for headings, important information, and emphasis

Font weights can be applied using tailwind classes:

```html
<span class="font-bold">
  This is bold text
</span>

<span class="font-semibold">
  This is semibold text
</span>

<span class="font-light">
  This is light text
</span>

```

## Text Styles

Our typography system is organized into several categories:

- **Article styles** - Article related stiles like headings, types, quotes, body, drop caps
- **Body text styles** - Standard body text styles
- **Headings** - Multiple headings for other pages than articles
- **Meta text** - Meta information like author, date, etc.
