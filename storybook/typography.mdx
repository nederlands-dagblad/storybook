## Responsive Typography

Our typography system supports responsive sizing across different breakpoints:
- **Mobile**: Default styles
- **Tablet**: 768px and above
- **Desktop**: 1024px and above

You can use our responsive typography components to automatically adjust text size, line height, and letter spacing based on screen size:

```jsx
import { ResponsiveHeading, ResponsiveBody } from '../components/Typography/ResponsiveTypography';

function Article() {
  return (
    <div>
      <ResponsiveHeading>Article Title</ResponsiveHeading>
      <ResponsiveBody>Article content goes here...</ResponsiveBody>
    </div>
  );
}
```

## Usage

You can use our typography components directly:

```jsx
import {
  HeadingArticle,
  BodyArticle,
  MetaAuthor
} from '../components/Typography/Typography';

function Article() {
  return (
    <div>
      <HeadingArticle>Article Title</HeadingArticle>
      <MetaAuthor>By John Doe</MetaAuthor>
      <BodyArticle>Article content goes here...</BodyArticle>
    </div>
  );
}
```

Or use the utility classes directly in your HTML:

```html
<h1 class="text-headings-heading-article">Article Title</h1>
<span class="text-meta-author">By John Doe</span>
<p class="text-body-body-article">Article content goes here...</p>
```
