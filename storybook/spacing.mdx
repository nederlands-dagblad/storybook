# Spacing

In our design system, we use t-shirt sizes to define spacing. Here are the sizes we use:
- `xs`: Extra small
- `s`: Small
- `m`: Medium
- `l`: Large
- `xl`: Extra large

<div style={{ marginTop: '2rem' }}>
    {(() => {
        const spacing = tailwindConfig.theme.extend.spacing;

        const entries = Object.entries(spacing).map(([name, value]) => (
            <div key={name} style={{ marginBottom: '2rem', padding: '1rem', border: '1px solid #e0e0e0', borderRadius: '4px' }}>
                <div style={{ display: 'flex', alignItems: 'center', gap: '1rem', marginBottom: '0.5rem' }}>
                    <strong style={{ minWidth: '60px' }}>{name}</strong>
                    <code style={{ background: '#f5f5f5', padding: '0.25rem 0.5rem', borderRadius: '3px' }}>
                        {value}
                    </code>
                </div>
                <div style={{ background: '#f9f9f9', padding: '0.5rem', borderRadius: '3px' }}>
                    <div style={{
                        width: value,
                        height: '32px',
                        background: '#0096cb',
                        borderRadius: '2px'
                    }}></div>
                </div>
            </div>
        ));

        return <div>{entries}</div>;
    })()}
</div>

## Usage

Spacing can be used for padding, margin, or gaps between elements:
```css
/* Padding */
.p-xs { padding: var(--spacing-xs); }

/* Margin */
.m-s { margin: var(--spacing-s); }

/* Gap */
.gap-m { gap: var(--spacing-m); }
```

For more information, read the Tailwind documentation:
- [Padding](https://tailwindcss.com/docs/padding)
- [Margin](https://tailwindcss.com/docs/margin)
- [Gap](https://tailwindcss.com/docs/gap)
