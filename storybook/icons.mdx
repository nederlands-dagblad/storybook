# Icons

On this page, you can find all icons available in our design system.

### Fill

All icons which are solid/filled.

<IconGallery>
  {keys('fill').map(iconName => (

      <Icon name={iconName} />
    </IconItem>
  ))}
</IconGallery>

### Outline

All icons which are outlined.

<IconGallery>
    {keys('outline').map(iconName => (
        <IconItem key={iconName} name={iconName.replace('-outline', '')}>
            <Icon name={iconName} />
        </IconItem>
    ))}
</IconGallery>

### Note on adding new icons

To add a new icon, you need to add the SVG file to the `src/assets/icons` folder. Icons are separated by variation: `outline` or `fill`.

After adding the SVG file, add the both variations to `src/components/Icon/list.ts`. This file contains a list of all available icons in the design system.

Finally, you can use the icon in your components by importing the `Icon` component and passing the name of the icon as a prop.
