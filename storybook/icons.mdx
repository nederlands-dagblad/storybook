# Icons

On this page, you can find all icons available in our design system.

### Fill

All icons which are solid/filled.

<IconGallery>
  {getIconNames().map(iconName => (
    <IconItem key={iconName} name={iconName}>
      <Icon name={iconName} variant="fill" />
    </IconItem>
  ))}
</IconGallery>

### Outline

All icons which are outlined.

<IconGallery>
    {getIconNames().map(iconName => (
        <IconItem key={iconName} name={iconName}>
            <Icon name={iconName} variant="outline" />
        </IconItem>
    ))}
</IconGallery>

### Note on adding new icons

To add a new icon, you need to add the SVG file to the `src/assets/icons` folder. Icons are separated by variation: `outline` or `fill`.

After adding the SVG file, add both variations to `src/components/atoms/basicAtoms/Icon/list.ts`. This file contains a list of all available icons in the design system.

You can find the icon component [here](?path=/docs/atoms-basic-atoms-icon--docs).
